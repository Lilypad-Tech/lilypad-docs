# Run "Hello, World!" Job

## Run Hello, :cow: World! Job

Start Docker

Open a Terminal window and run the following command:

```bash
lilypad run --template cowsay:v0.0.1 --params "hello lilypad"
```

(ensure your user is in the docker group if necessary on your platform)\
